- **Display:** 24px / 32px / 600
- **Title:** 20px / 28px / 600
- **Body:** 15px / 22px / 500
- **Body Small:** 13px / 18px / 500
- **Caption:** 11px / 16px / 600

## Spacing Scale
- 4, 8, 12, 16, 20, 24, 32, 40

## Corner Radius
- **Small:** 12px
- **Medium:** 16px
- **Large:** 24px

## Elevation Levels

- **Level 2:** `0 8px 24px rgba(16,24,40,0.10)`
- **Level 3:** `0 16px 40px rgba(16,24,40,0.12)`

## Icon Style
- Lucide/outlined icons via Iconify, 20px default, rounded stroke.

## Component Rules
- Buttons: 44px min height, 16px radius, bold labels.
- Inputs: 48px height, visible focus ring.
- Cards: 24px radius, consistent padding, subtle shadow.
- List rows: 12px vertical padding, 16px horizontal padding, one-line title + optional subtitle.
- Badges: 18px height, uppercase or short labels.
- Navigation: fixed top bar + bottom nav on mobile, consistent icons.
